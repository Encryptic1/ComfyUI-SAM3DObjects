# Assets

This directory will contain example images, masks, and other assets for testing and demonstration.

## Planned Assets

- `example_images/` - Sample input images
- `example_masks/` - Corresponding segmentation masks
- `example_rgba/` - RGBA images with alpha masks
- `example_outputs/` - Example 3D outputs (PLY, OBJ files)

## Usage

Assets will be provided for:
- Testing the nodes
- Learning workflows
- Demonstrating capabilities

Example assets will be added in future updates.
